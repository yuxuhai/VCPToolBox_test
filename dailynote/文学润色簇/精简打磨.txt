【精简打磨】
删繁就简的艺术

精简原则：
每个字都要有存在价值

删减对象：
1. 冗余词汇
   - 重复表达
   - 可有可无的修饰
   - 空洞的形容词

2. 啰嗦句式
   - 能用1句说清别用3句
   - 去掉"的"字过多
   - 简化从句

3. 无效信息
   - 不推动情节
   - 不刻画人物
   - 不营造氛围

4. 说明性语言
   - 直接告诉读者的
   - 可以展现的别说明

精简技巧：
• 动词替代
  "快速地跑" → "冲"
  "大声说" → "吼"

• 去掉修饰堆砌
  "非常非常漂亮" → "惊艳"

• 合并短句
  多个短句 → 精炼长句

• 删掉废话
  "他想了想" → 直接说想的

• 具体代替抽象
  "很好" → 具体好在哪

检查清单：
□ 每个形容词必要吗？
□ 这句话能更短吗？
□ 这段话能删掉吗？
□ 能展现的别告诉？

示例对比：
冗余：他非常非常愤怒地用手狠狠地砸向了桌子
精简：他砸向桌子

啰嗦：她是一个很漂亮的、有着长长头发和大大眼睛的女孩
精简：长发、大眼，很美

打磨流程：
初稿 → 删减30% → 再删10% → 完成

注意：
✓ 精简≠干瘪
✓ 保留必要细节
✓ 保持韵律感
✓ 该详写处详写

避免：
✗ 过度删减失去味道
✗ 删掉关键信息
✗ 机械追求字数少

输出：
- 冗余识别：[可删部分]
- 精简策略：[如何简化]
- 核心保留：[必须留的]
- 优化效果：[简化后]
