【对话塑造】
通过对话展现角色个性

对话功能：
1. 推进情节
2. 展现性格
3. 揭示关系
4. 传递信息
5. 营造氛围

个性化对话：
• 说话方式
  - 用词：粗俗/文雅/专业
  - 句式：短/长/破碎
  - 语气：强硬/温柔/戏谑

• 说话内容
  - 直接/委婉
  - 多/少
  - 真话/谎言

• 说话习惯
  - 口头禅
  - 重复某些词
  - 特殊语调

对话技巧：
✓ 每个角色声音不同
✓ 潜台词丰富
✓ 有冲突有张力
✓ 避免信息dump

潜台词：
表面说A，实际想说B

示例：
"你今天真漂亮"
潜台词可能是：
- 以前不漂亮（讽刺）
- 有所图（讨好）
- 真心赞美（善意）

对话节奏：
• 问答交替
• 长短搭配
• 打断、沉默
• 情绪起伏

对话禁忌：
✗ 所有人说话一样
✗ 直白说明性信息
✗ 无意义闲聊
✗ 完整正式句子（太假）

真实对话：
→ 有停顿
→ 有重复
→ 有口误
→ 有省略
→ 有语气词

情绪对话：
愤怒：短促、重复、夸张
悲伤：断续、低语、自嘲
紧张：语无伦次、快速
亲密：省略、暗示、共同语言

输出：
- 对话风格：[角色说话特点]
- 关键对话：[重要对话场景]
- 潜台词：[言外之意]
- 关系展现：[通过对话显示]
