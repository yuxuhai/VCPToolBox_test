【结尾设计】
完美收束故事

结尾重要性：
最后印象
决定故事整体评价

有效结尾类型：
1. 完结式结局
   所有问题解决
   "从此幸福生活"
   
   适合：童话、浪漫喜剧

2. 开放式结局
   留有空间想象
   答案不明确
   
   适合：文艺片、反思作品

3. 转折式结局
   最后反转
   重新理解一切
   
   适合：悬疑、惊悚

4. 圆环式结局
   回到开头场景
   但意义不同
   
   暗示循环或成长

5. 余韵式结局
   不刻意总结
   自然淡出
   
   文学性强

结尾要素：
✓ 高潮解决
  核心冲突处理

✓ 角色成长展现
  证明转变

✓ 主题升华
  点明意义

✓ 情感满足
  给读者所需

✓ 新常态
  展现改变后世界

结尾避免：
✗ Deus Ex Machina
  神仙降临解决

✗ 梦醒结局
  "原来是梦"

✗ 过度解释
  不信任读者

✗ 虎头蛇尾
  草草收场

✗ 新情节突现
  最后引入新元素

结尾节奏：
• 高潮：快速激烈
• 降落：逐渐平复
• 结尾：适当篇幅

不要高潮后立即结束
给情感消化空间

悲剧vs喜剧结局：
• 喜剧
  - 问题解决
  - 团圆或成功
  - 正面情感

• 悲剧
  - 主角失败
  - 但有所领悟
  - 深刻意义

苦乐参半：
最真实的结局
- 有得有失
  有成长有代价
- 复杂情感

最后一句：
像首句一样重要
- 回响余韵
- 点题升华
- 情感冲击

结尾测试：
✓ 是否满足承诺？
✓ 角色弧光完整？
✓ 主题是否清晰？
✓ 情感是否满足？

输出：
- 结局类型：[哪种结局]
- 核心解决：[如何收束]
- 情感基调：[最终感受]
- 余韵：[留给读者什么]
