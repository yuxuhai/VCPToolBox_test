【历史背景】
为世界建立时间深度

历史重要性：
历史塑造现在
解释为什么是这样

历史构建层次：
1. 创世神话
   - 世界起源
   - 神话传说
   - 文化根基

2. 古代历史
   - 文明兴衰
   - 重大战争
   - 英雄事迹

3. 近代历史
   - 社会变革
   - 科技进步
   - 政治演变

4. 当代背景
   - 故事前10-50年
   - 直接影响现状

历史设计原则：
✓ 服务故事
  只设计相关历史

✓ 因果清晰
  历史事件有因果

✓ 影响现在
  过去塑造现在

✓ 留有空间
  不必面面俱到

关键历史事件：
设计影响深远的事件：
- 战争革命
- 灾难瘟疫
- 重大发明
- 社会变革

《冰与火之歌》：
"疯王"被杀→现在权力格局

历史与角色：
• 个人历史
  角色家族历史
  塑造角色

• 代际传承
  祖辈的选择影响后代

• 历史创伤
  集体记忆影响社会

历史展现技巧：
→ 通过对话提及
→ 遗迹废墟暗示
→ 节日仪式纪念
→ 历史书籍文献
→ 老人回忆讲述

避免：
✗ 大段历史讲解
✗ 与故事无关的历史
✗ 历史矛盾

历史时间线：
```
5000年前：创世神话
3000年前：古代文明
1000年前：帝国崩溃
500年前：新秩序建立
100年前：科技革命
50年前：最近战争
现在：故事开始
```

- 历史中寻找答案
