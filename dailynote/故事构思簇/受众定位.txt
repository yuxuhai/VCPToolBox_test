【受众定位】
明确目标读者并针对性创作

受众分析维度：
1. 年龄段
   - 儿童（6-12岁）：简单清晰，道德明确
   - 青少年（13-17岁）：成长、身份、叛逆
   - 青年（18-35岁）：爱情、事业、探索
   - 中年（36-55岁）：家庭、责任、反思
   - 老年（55+）：回忆、智慧、传承

2. 性别偏好
   - 避免刻板印象
   - 关注普遍人性
   - 多元化视角

3. 文化背景
   - 价值观差异
   - 文化符号
   - 本地化考虑

4. 阅读习惯
   - 通俗vs文艺
   - 快节奏vs慢叙事
   - 娱乐vs深度

受众画像创建：
• 人口统计
  年龄、性别、教育、职业

• 心理特征
  价值观、兴趣、痛点、渴望

• 阅读偏好
  喜欢的类型、作者、作品

示例画像：
```
核心受众：25-35岁都市女性
特征：
- 受过高等教育
- 职场白领
- 关注情感与成长
- 喜欢现实主义题材
- 追求情感共鸣
```

针对性创作策略：
→ 主题选择：符合受众关切
→ 复杂度：匹配认知水平
→ 叙事节奏：适应阅读习惯
→ 语言风格：贴近受众表达

年龄适配示例：
• 儿童读物
  - 主角是儿童
  - 语言简单
  - 情节直接
  - 正面结局

• 成人文学
  - 复杂人物
  - 道德灰度
  - 多重主题
  - 开放结局

跨龄吸引：
最好的故事能吸引多年龄层
《哈利·波特》：
- 儿童：魔法冒险
- 青少年：成长认同
- 成人：政治隐喻

验证方法：
✓ 测试阅读：找目标受众试读
✓ 同类对比：分析同类作品
✓ 数据分析：市场反馈

输出：
- 核心受众：[详细画像]
- 吸引策略：[如何吸引]
- 内容调整：[针对性设计]
- 跨界潜力：[扩展可能]
