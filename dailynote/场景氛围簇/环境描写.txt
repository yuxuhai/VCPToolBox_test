【环境描写】
构建生动的场景空间

描写要素：
1. 视觉元素
   - 色彩：明暗、冷暖
   - 形状：大小、轮廓
   - 光影：光源、阴影
   - 布局：空间、位置

2. 听觉元素
   - 自然声：风、雨、虫鸣
   - 人造声：机械、人声
   - 节奏：急促、舒缓
   - 音量：响亮、微弱

3. 触觉元素
   - 温度：冷、暖、炽热
   - 质感：粗糙、光滑
   - 湿度：干燥、潮湿

4. 嗅觉/味觉元素
   - 气味：花香、腐臭、烟味
   - 味道：空气的味道

描写原则：
• 选择性描写：重要细节
• 感官化描写：调动五感
• 情绪化描写：与氛围契合
• 功能性描写：为故事服务

描写技巧：
✓ 从特定视角观察
✓ 动静结合
✓ 点面结合
✓ 虚实结合

环境功能：
→ 交代背景
→ 渲染氛围
→ 暗示情节
→ 映衬心情
→ 推动发展

示例对比：
平淡：房间很乱
生动：衣服堆在椅子上，咖啡渍在桌面结成褐色的圈，窗帘半掩着灰蒙蒙的天光。

避免：
✗ 流水账式罗列
✗ 静止的说明
✗ 无关紧要的细节
✗ 过度描写

输出：
- 核心场景：[主要环境]
- 感官细节：[视听嗅触]
- 氛围基调：[整体感受]
- 功能作用：[为何这样写]
