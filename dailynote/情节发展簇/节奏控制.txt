【节奏控制】
掌握故事的紧张与舒缓

节奏定义：
故事推进的速度
情节密度与情感强度

节奏类型：
• 快节奏
  - 短场景
  - 快速对话
  - 持续行动
  - 高强度冲突
  
  适合：动作、惊悚、悬疑

• 慢节奏
  - 长场景
  - 细节描写
  - 内心独白
  - 氛围营造
  
  适合：文艺、哲学、史诗

节奏波动：
最好的故事不是单一节奏
而是有起伏的波浪

```
强度
 ↑    /\      /\
 │   /  \    /  \  高潮
 │  /    \  /    \
 │ /      \/      \
 └─────────────────→ 时间
   快慢快  慢  快
```

场景节奏：
• 动作场景(Scene)
  - 实时发生
  - 对话、冲突
  - 推进情节
  - 快节奏

• 过渡场景(Sequel)
  - 事后反应
  - 情感消化
  - 决策思考
  - 慢节奏

场景-过渡循环：
Scene → Sequel → Scene → Sequel

句子层面节奏：
→ 短句：快节奏、紧张
  "他跑。门开。枪响。"

→ 长句：慢节奏、舒缓
  "他慢慢走过长廊，脑海中浮现出童年的记忆..."

段落层面节奏：
• 短段落：快速推进
• 长段落：详细描述

调节节奏技巧：
1. 改变场景长度
   紧张时：短场景连续
   舒缓时：长场景展开

2. 对话vs描述比例
   对话：快
   描述：慢

3. 时间跳跃
   快进非重点部分

4. 视角切换
   悬念章节结尾切换

5. 子情节交织
   主线紧张时插入轻松支线

关键节奏点：
✦ 开篇：快速吸引
✦ 第一幕：稳步建立
✦ 第二幕前半：加速
✦ 中点：高峰
✦ 第二幕后半：最快
✦ 高潮前：短暂平静
✦ 高潮：爆发
✦ 结局：逐渐平复

避免节奏问题：
✗ 过快：读者疲劳，缺少情感
✗ 过慢：读者无聊，失去兴趣
✗ 单调：没有起伏变化

输出：
- 整体节奏：[快/中/慢及变化]
- 场景节奏：[各场景速度]
- 调节手法：[如何控制]
- 关键高峰：[情感爆发点]
