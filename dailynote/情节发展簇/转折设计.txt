【转折设计】
创造意外而合理的情节转折

转折定义：
情节的意外转向
打破读者预期

转折作用：
• 制造惊喜
• 重新吸引注意
• 加深主题
• 推动情节

有效转折特征：
✓ 意外但合理
  - 出乎意料
  - 回顾时有迹可循

✓ 改变一切
  - 重新定义前情
  - 改变后续走向

✓ 情感冲击
  - 震惊、恐惧、感动
  - 留下深刻印象

转折类型：
1. 身份转折
   原来他是___
   《第六感》：主角已死
   《星球大战》：父子关系

2. 动机转折
   原来他想要的是___
   真实目的揭示

3. 情境转折
   原来情况是___
   《搏击俱乐部》：人格分裂

4. 道德转折
   正派变反派/反派变正派
   角色立场反转

5. 计划转折
   A计划失败→启动B计划
   或：看似失败实为成功

关键转折点位置：
→ 第一幕结尾（25%）
  常规世界被打破

→ 中点（50%）
  假胜利或假失败
  信息揭示改变认知

→ 第二幕结尾（75%）
  至暗时刻
  最大转折

→ 高潮（90%）
  最终反转

铺垫技巧：
• 切赫夫的枪
  "第一幕出现的枪，第三幕必须响"
  
  提前埋下伏笔
  但不要太明显

• 红鲱鱼（障眼法）
  误导读者注意力
  真相在别处

• 双重意义
  台词/场景有两层解读
  转折后重新理解

设计转折步骤：
1. 确定常规预期
   读者会认为___

2. 设计意外真相
   实际上是___

3. 合理化证明
   回顾细节能发现线索

4. 最大化冲击
   选择最佳揭示时机

示例设计：
```
设定：侦探追捕连环杀手
预期：找到凶手并逮捕
转折：侦探发现自己就是凶手（人格分裂）
铺垫：
- 侦探记忆空白
- 目击者描述疑似侦探
- 凶手了解案件细节
```

避免糟糕转折：
✗ 完全随机，无法预见
✗ 过于复杂，难以理解
✗ 降低故事，纯为惊奇
✗ 不改变任何东西

输出：
- 主要转折：[关键转折点]
- 转折时机：[在哪里]
- 铺垫线索：[如何埋伏笔]
- 情感冲击：[预期效果]
