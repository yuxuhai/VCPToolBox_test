【叙事视角】
选择合适的讲述角度

视角类型：
1. 第一人称（我）
   优势：
   ✓ 亲密感强
   ✓ 情感真实
   ✓ 代入感好
   
   局限：
   ✗ 信息受限
   ✗ 无法全知
   ✗ 难写群像

2. 第三人称全知
   优势：
   ✓ 视角自由
   ✓ 信息完整
   ✓ 可评论
   
   局限：
   ✗ 距离感
   ✗ 易说教
   ✗ 缺少神秘

3. 第三人称限知
   优势：
   ✓ 灵活适中
   ✓ 可深入内心
   ✓ 保持客观
   
   特点：
   - 跟随特定角色
   - 知其所知
   - 感其所感

4. 第二人称（你）
   特点：
   - 罕见但有力
   - 强制代入
   - 实验性强

视角选择：
• 悬疑推理：第一人称或限知
• 史诗群像：全知视角
• 心理深度：第一人称
• 多线并行：多视角切换

视角切换：
单一视角：始终如一
多重视角：章节切换不同POV
交替视角：两个主角轮流

注意事项：
✗ 视角混乱跳跃
✗ 突破视角限制
✗ 频繁切换晕眩

技巧：
→ 明确视角规则
→ 保持一致性
→ 利用视角优势
→ 弥补视角缺陷

输出：
- 主要视角：[采用什么]
- 视角原因：[为何选择]
- 视角限制：[不能知道什么]
- 信息策略：[如何传递]
